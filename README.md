# Diagnóstico de Falla en Motor Caterpillar C15
Este proyecto simula un caso real de **diagnóstico de fallas en un motor diésel Caterpillar C15**, ampliamente utilizado en minería y transporte pesado.   El objetivo es identificar una **falla incipiente** mediante el análisis de datos operativos y la aplicación de criterios técnicos de mantenimiento y confiabilidad.
